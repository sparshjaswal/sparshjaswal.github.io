# IDENTIFIERS & KEYWORDS

**Identifiers** _are the name assigned by the user of the program using certain keywords._

**Keywords** _are the set of words that have special meaning to compilers._

#### _Below are all keywords that are currently supported by the latest JavaScript standards_
