## How it Started

After Netscape made the first commercial web browser called Netscape Navigator. It was very popular and a huge hit. The Netscape team wanted to improve and introduce new features like E-commerce, better look, easy for end-user programming and something similar to HyperCard of macintosh to be implemented within Netscape 2.

**Brend Eich** was hired for this role and created a new language in 10 days which includes syntax & concept from java, scheme, and self. This language was named as LiveScript.At the same time, Sun and Netscape jointly ventured and renamed it to JavaScript.
ECMAScript name is compromised between Sun, ECMAScript, and Netscape for standardizing.

Key Feature of Javascript
Load and go delivery
Loose typing
Objects as general containers
Prototypal inheritance
Lambda - function as a first-class object
Linkage through global variables
JavaScript is inspired by Scheme(functional programming) and Java (object-based language)

## What is JavaScript?

> JavaScript (JS) is a lightweight, interpreted or just-in-time compiled programming language with first-class functions.JavaScript is a prototype-based, multi-paradigm, dynamic language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles

*Note:- JavaScript has no significant relationship with the java.*

## What is a scripting language? Is JavaScript a scripting language?

Theoretically, a **scripting language** doesn’t do a compilation step i.e there is no intermediate code/compilations code like in c,c++ or java does. Scripting languages are directly interpreted.

Yes, JavaScript is a scripting language but to improve the performance and executions of javaScript code it is compiled using techniques like JIT( just in time compiler) or AOT(ahead of time) is the most common browser’s  JSC (javascript compiler) like V8(chrome), Mozilla(spider monkey), Karma(Microsoft).
