## Activity login Page
##### Below is expected from Activity 
 1. Page should contain 
     1. User name field
     1. Password field
     1. Login Button
     
 1. Username must follow below Rule
    1. Username should be inbetween 8 to 13 letter long
    1. Usernames can contain only letters (a-z), numbers (0-9), and periods (.)
    1. Usernames can begin or end with periods (.)
 
 1. Password must follow below Rule
    1. Password should be inbetween 8 to 13 letter long
    1. Spaces are not allowed at the beginning and end of passwords
    1. Must contain at least 
         1. one lower case letter
         1. one upper case letter
         1. one digit
         1. one of these special characters
         
 1. After entering username validate.
 1. After entering password validate.
 1. If the username and password are correct then move to the next page.
 
##### Other Supported features
 1. login should support Key navigation
